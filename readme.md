## Installation/Running instructions:
This program runs on Python 3.x. In order to run it, open a terminal,
navigate to the directory containing minimax.py, and run the following command:
1. On Linux:
`./minimax.py`
2. On Windows: 
`python3 minimax.py`
3. On Mac:
`python3 ./minimax.py`
